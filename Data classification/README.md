## Project Overview

This project uses Machine Learning to predict whether an employee is likely to leave the company or stay. It trains multiple classification models and compares their performance to find the most accurate one.

## Dataset
File: employee.csv
Target Column: Attrition
Problem Type: Binary Classification
Yes = Employee left the company
No = Employee stayed in the company

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn

## Models Used
Decision Tree Classifier
Random Forest Classifier
Logistic Regression

## Project Workflow
Load the dataset
Explore and preprocess the data
Encode categorical features
Split data into training and testing sets
Train machine learning models
Evaluate model accuracy
Select the best-performing model

## How to Run

Install the required libraries:

pip install pandas numpy matplotlib scikit-learn

Run the project:

python attrition_prediction.py

## Output

The program:

Displays dataset information
Shows employee attrition distribution
Prints the accuracy and classification report of each model
Identifies the best model
Displays an accuracy comparison chart
