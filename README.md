# Rule-Based AI Chatbot

## Description
A simple rule-based chatbot developed using Python in Visual Studio Code (VS Code). The chatbot responds to predefined user inputs using if-else statements and runs continuously until the user exits.

## Features
- Greeting responses
- AI and Python information
- Date and time display
- Exit command
- Continuous conversation loop

## Tools Used
- Python 3
- Visual Studio Code (VS Code)

## How to Run
1. Open the project in VS Code.
2. Open the terminal.
3. Run:

python chatbot.py
